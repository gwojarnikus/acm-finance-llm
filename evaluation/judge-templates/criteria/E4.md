# Criterion E4: Output Length Discipline

## Definition

Is the report's length proportionate to the complexity of the question? No padding (the well-documented LLM tendency to verbose responses, especially under RLHF training).

This criterion explicitly counters the verbosity bias in LLM-as-Judge: research has shown that LLM judges systematically prefer longer responses regardless of substance (Saito, Wachi, Wataoka, & Akimoto, 2023; Hu et al., 2025). E4 forces the judge to score AGAINST that tendency.

## CRITICAL: verbosity self-warning

You (the LLM judge) have a measurable tendency to prefer longer responses (Saito et al., 2023). For E4 specifically:

- DO NOT equate "longer" with "better". 
- Penalize length without substance.
- If you find yourself thinking "this is thorough because it's detailed", check: is the detail informative, or is it padding (repetition, generic context-setting, academic hedging)?

## Behavioral anchors (from BARS pilot v1)

- **Level 2 (poor):** Extremely short (<500 words for an analytical question) OR extremely long (>8000 words for a moderately complex question) without justification.
- **Level 5 (average):** Within range, but visibly padded with academic filler. Observed in R1 (e.g., S2-R1-G word_count ~3600 + "verbose academic style" mid-section — within range but verbose at times).
- **Level 8 (good):** Adequate length (per-case 2500–5000 words), low redundancy. Observed in R1 (e.g., S1-R1-C ~5600 words for a complex case (PL tax + EU law) — upper end but justified).
- **Level 10 (excellent):** Optimal length; each section proportional to topic weight. Aspirational.

Reports significantly outside these ranges (without strong justification) lose points.

## Scoring method

**Type:** length measurement + self-warning + qualitative density assessment.

**Procedure:**

1. **Measure word count** of the report. For R1, you may use `output_word_count_reported` as a reference but verify with body count if discrepancy is large.
2. **Range check:**
   - In expected range (2500–5000): baseline 7
   - Below 2500: -2 per 500 words below (penalty for insufficient analysis)
   - Above 5000: -1 per 1000 words above unless density is high (penalty for padding)
3. **Density adjustment (qualitative):**
   - +1 if every section delivers substantive content (no padding)
   - +1 if explicit prioritization (Bottom Line, key takeaways)
   - -1 if visible padding (repetition, generic context-setting)
4. **Verbosity self-check.** Before finalizing, ask: would removing 500 words from this report lose substantive content? If no, the report is padded — apply -1.

Score guidance:
- Optimal length + high density + prioritization → 9–10
- In range + high density → 7–8
- In range + mixed density → 5–6
- Outside range moderately + low density → 3–4
- Far outside range or extreme padding → 0–2

## Asymmetry per response type

- **AKM:** Moderator role redacts; expect tight final report.
- **B1:** single-shot; variable.
- **R1:** observed pilot — R1-G systematically longer than R1-C without proportional substance. Penalize R1-G length if not matched by density.
