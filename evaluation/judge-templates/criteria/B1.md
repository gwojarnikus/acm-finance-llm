# Criterion B1: Multi-dimensional Decomposition

## Definition

Does the report decompose the problem into a sufficient number of analytically meaningful dimensions (macro, micro, legal, technical, behavioral, regional, distributional)? A one-dimensional analysis is rarely adequate for complex financial-economic policy questions.

## Behavioral anchors (from BARS pilot v1)

- **Level 2 (poor):** Single dimension only (e.g., for S1, only the fiscal-arithmetic dimension; for S2, only portfolio mathematics). Other relevant angles missing.
- **Level 5 (average):** 2–3 dimensions present but each treated superficially (one paragraph each, no depth).
- **Level 8 (good):** ≥4 dimensions with moderate depth, each as a named section or clearly delimited block. Observed in R1 (e.g., S1-R1-C addresses: fiscal/arithmetic, sectoral/economic, legal/EU law, distributional, public-finance architecture, cascading supply-chain — 6+ dimensions).
- **Level 10 (excellent):** ≥5 dimensions with depth AND explicit reflection on their interactions ("the demographic dimension constrains the fiscal one because..."). Observed in R1 (e.g., S3-R1-C integrates fiscal + demographic + labour + gender + regional + informality + political economy with explicit cross-references like "That demographic point has a direct social dimension").

## Scoring method

**Type:** structural analysis with per-case minimums.

**Per-case minimum expected dimensions** (the report should cover at least these; failure to address them caps the score at ≤5):

| Case | Minimum dimensions |
|------|---------------------|
| **S1** (PL turnover tax) | fiscal-arithmetic, sectoral/economic impact, EU law/regulatory |
| **S2** (pension fund allocation) | asset-liability matching (ALM), macro environment, regulatory |
| **S3** (Italy UBI) | fiscal sustainability, demographic, labour market |

**Procedure:**

1. **Identify analytical dimensions** in the report. Look for named sections, subsections, or clearly delimited analytical perspectives. Each dimension addresses a distinct analytical angle (not just sub-aspects of one angle).
2. **Check per-case minimums.** Are all the expected dimensions covered substantively? "Mentioned in one sentence" does not count as covered.
3. **Count additional dimensions** beyond the minimum.
4. **Assess depth.** For each dimension, is there analysis (data + reasoning + conclusion) or just a passing reference?
5. **Look for interaction analysis.** Does the report explicitly reason about how dimensions affect each other?

Score guidance:
- All minimums covered + ≥2 additional dimensions + interaction analysis → 8–10
- All minimums covered + 1 additional + some depth → 6–7
- All minimums covered, but superficial → 5
- One or more minimum dimensions missing → 0–4

## Asymmetry per response type

- Apply uniformly. All three response types had access to the same case question and should decompose comparably.
