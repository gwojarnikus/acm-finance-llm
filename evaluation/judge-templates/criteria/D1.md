# Criterion D1: Operational Actionability

## Definition

Are the recommendations implementable? In business, value comes only from advice that can be acted upon. A recommendation like "consider improving capital structure" is not actionable; "issue €500M of 5-year senior unsecured debt at expected coupon 4.5%, refinancing the 2027 maturity" is actionable.

## Behavioral anchors (from BARS pilot v1)

- **Level 2 (poor):** "Should be considered", "needs further analysis" — generalities without parameters.
- **Level 5 (average):** Direction indicated ("increase inflation hedging"), but no concrete parameters (no amount, no timeframe, no instrument).
- **Level 8 (good):** Concrete recommendations with parameters (what, how much, when, which instrument). Observed in R1 (e.g., S2-R1-C lines 57–69: full portfolio allocation with percentages — 25% linkers, 15% short-duration, 25% equities, 15% infra, 8% real estate, 7% commodities, 5% FRN — plus rationale per sleeve).
- **Level 10 (excellent):** Plus implementation plan with phases and success metrics. Observed in R1 (e.g., S2-R1-C line 73: `four implementation rules: measure success against real funded status, hold 12 months cash buffer, hedge FX, rebalance with discipline` — explicit governance framework).

## Scoring method

**Type:** actionable items count + completeness check.

**Procedure:**

1. **Recommendation inventory.** Identify the recommendations in the report's Recommendation / Conclusions / Bottom Line section.
2. **Per-recommendation checklist.** For each recommendation, check 3 properties:
   - **What:** is the action specified concretely? ("issue debt" not "improve capital structure")
   - **How much:** is the magnitude specified? (amount, percentage, count, etc.)
   - **When:** is the timeframe specified? (immediately, by date, on trigger, etc.)
   Each property: 1 point.
3. **Aggregate:** sum points across recommendations; normalize to [0, 10].
4. **Bonus for implementation plan:** +1 if there is an explicit phased implementation framework.
5. **Bonus for success metrics:** +1 if the report specifies how to measure whether the implementation worked.

Score guidance:
- ≥3 recommendations with all 3 properties + implementation plan + success metrics → 9–10
- ≥3 recommendations with all 3 properties → 7–8
- Recommendations with 2 of 3 properties → 5–6
- Recommendations with 1 of 3 properties → 3–4
- Vague recommendations only → 0–2

## Asymmetry per response type

- Apply uniformly. All response types should produce actionable recommendations for case prompts.
- Note: S1 and S3 (policy questions) may have less concrete "amounts" than S2 (portfolio allocation, where numerical specificity is natural). Adjust expectations: for policy cases, specificity may be in legal mechanism details rather than amounts.
