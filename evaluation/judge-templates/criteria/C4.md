# Criterion C4: Insight Density

## Definition

Ratio of unique insights to text volume. Elimination of "filler", repetition, and generic statements. In professional finance reports, time is scarce — every paragraph must earn its place.

C4 differs from D7 (Information Density, linguistic) in focus: C4 measures *insight* density (substantive analytical claims per 1000 words); D7 measures *linguistic* density (conciseness, absence of filler words). Both can be assessed independently.

## Behavioral anchors (from BARS pilot v1)

- **Level 2 (poor):** Filler; same theses repeated in different words; paragraphs that could be deleted without information loss.
- **Level 5 (average):** Mix of insights and repetitions; some paragraphs dense, others padded.
- **Level 8 (good):** Each sentence carries value; no generic statements like "this is important" or "more research is needed" without specifics. Observed in R1 (e.g., S1-R1-C line 49: short, dense paragraph with 4 datapoints + conclusion + citation in one paragraph).
- **Level 10 (excellent):** Condensed, executive-grade; explicit prioritization of insights (e.g., "Bottom Line" section with bulleted key conclusions). Aspirational.

## Scoring method

**Type:** insights-per-kword ratio + qualitative judgment.

**Procedure:**

1. **Word count.** Estimate the report's word count (or use `output_word_count_reported` for R1 — note: ignore the self-reported number in scoring, use it only for context).
2. **Unique insights count.** Count distinct substantive analytical claims. A "unique insight" is:
   - A novel inference from data (not just data restatement)
   - A specific recommendation with parameters
   - A counter-intuitive observation explained
   - A causal mechanism articulated
   - A comparison with quantified outcomes
3. **Ratio computation.** Compute insights per 1000 words. Reference points from R1 pilot:
   - R1-C reports: ~5600 words, ~50 unique insights → ~9 insights/kw → moderate-high density
   - R1-G reports: ~3600 words, ~30 unique insights → ~8 insights/kw → moderate (more padding)
4. **Quality adjustment:**
   - Subtract for visible filler ("In conclusion", "It is important to note", paragraphs that just summarize the previous one).
   - Add for explicit prioritization (e.g., "Bottom Line", "Key Takeaways" with bulleted points).

Score guidance:
- Insights/kw ≥ 10 + explicit prioritization → 9–10
- Insights/kw 7–10, low filler → 7–8
- Insights/kw 5–7, mixed → 5–6
- Insights/kw 3–5, visible padding → 3–4
- Insights/kw < 3, mostly filler → 0–2

## Note on verbosity bias

Per grounding spec §5.1: you have a tendency to favor longer responses. For C4 specifically, this means you may over-credit a long report with many insights even if the same insights could be expressed in half the space. Counter-check: would a 1500-word version of this report lose substantive content? If no, the report is padded.

## Asymmetry per response type

- **AKM:** the Moderator agent's role includes redaction. If functioning, expect higher C4 — the final report should be condensed by Moderator.
- **B1:** single-shot, no redaction step. Expect variable C4.
- **R1:** Deep Research often verbose academic style. Expect moderate C4, with R1-G systematically more verbose than R1-C.
