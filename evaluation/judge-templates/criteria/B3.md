# Criterion B3: Cognitive Bias Mitigation

## Definition

Is the report free from systematic cognitive biases that LLMs tend to inherit from training data and exhibit when reasoning under uncertainty? Specifically: confirmation bias, anchoring bias, recency bias, availability heuristic, and survivorship bias.

This criterion is intentionally hard for LLM judges — assessing biases requires meta-cognitive reflection. The methodology below uses pre-defined behavioral patterns (rather than open-ended assessment) to make this tractable.

## Behavioral anchors (from BARS pilot v1)

- **Level 2 (poor):** Clear confirmation bias — only sources supporting the thesis are cited; no alternative perspectives engaged.
- **Level 5 (average):** Some biases present (e.g., recency bias — dominance of <12-month sources), but counterargument section exists. Observed in R1 (e.g., S2-R1-G dominated by 2024-2026 sources but engages with foundational 2004 Gorton & Rouwenhorst).
- **Level 8 (good):** Active mitigation of ≥2 biases via concrete moves: counterargument construction + citation of both recent and foundational works + weighing of different schools of thought. Observed in R1 (e.g., S3-R1-C balances "UBI might dramatically reduce poverty" steel-manning with empirical limits from Finland/Alaska experiments).
- **Level 10 (excellent):** Plus explicit meta-cognitive reflection on own biases + application of de-biasing techniques (pre-mortem analysis, devil's advocate, base rate consideration).

## Scoring method

**Type:** pattern check with pre-defined biases (2 points per pattern, 4 patterns = 8 base + 2 for meta-cognitive reflection = 10 max).

**Procedure:** For each of the 4 bias patterns below, score 0/1/2:
- 0 = bias evident in report
- 1 = bias partially mitigated
- 2 = bias actively mitigated

### Pattern 1: Confirmation bias

Check: does the report cite only sources supporting its thesis, or does it engage with contrary evidence?
- 0 = only supporting sources cited; opposing views ignored
- 1 = some opposing views mentioned but quickly dismissed
- 2 = opposing views steel-manned (best version of opposing argument presented), then engaged

### Pattern 2: Anchoring bias

Check: does the first numerical value mentioned disproportionately influence subsequent conclusions? Is there a tendency to round all estimates toward an initial anchor?
- 0 = clear anchoring (e.g., first estimate becomes the "center" and all others cluster around it without independent derivation)
- 1 = some anchoring but mitigated by independent calculations
- 2 = each estimate derived independently; no anchoring detected

### Pattern 3: Recency bias

Check: does the source list disproportionately favor very recent works (<6 months) while ignoring foundational older literature where relevant?
- 0 = all sources from <12 months; foundational works missing
- 1 = mix of recent and older sources, but ratio skewed
- 2 = appropriate mix; foundational citations present where relevant (e.g., for asset allocation: classic finance theory papers; for tax law: long-standing CJEU rulings)

### Pattern 4: Survivorship bias

Check: when discussing examples or case studies, are both successful and failed cases considered? Or only successes (e.g., only successful UBI pilots, only countries where a tax worked)?
- 0 = only successful cases cited; failures invisible
- 1 = mix but skewed toward successes
- 2 = balanced; explicit consideration of cases where the approach failed

### Final score

`score = sum(pattern_scores) + meta_cognitive_bonus`

Where:
- `sum(pattern_scores)` ∈ [0, 8]
- `meta_cognitive_bonus = 2` if the report explicitly discusses its own potential biases (e.g., "we may be subject to recency bias because..."); else 0.
- Final score capped at 10, rounded to 0.5.

## Asymmetry per response type

- **AKM:** the multi-agent architecture (Analyst-Critic-Moderator) is hypothesized to reduce biases through dialectical pressure. The Critic role explicitly targets biases. Expect higher B3 if architecture works as designed.
- **B1:** single-shot LLM more vulnerable to biases — less internal pressure to mitigate.
- **R1:** Deep Research training may include bias mitigation; baseline expectation moderate.
